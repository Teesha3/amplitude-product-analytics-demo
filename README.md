# Product Analytics Demo – Amplitude

This project is a simple demo landing page created to demonstrate basic product analytics implementation using Amplitude.

## Project Overview
I created a demo landing page with two CTA buttons:
- Sign Up
- Request Demo

Amplitude analytics was implemented to track user interactions and button click events.

## Analytics Implemented
- Event Tracking (CTA Clicks)
- Funnel Analysis
- User Behavior Segmentation

## Funnel Example
Landing Page → Sign Up Click → Request Demo Click

## Purpose
This project demonstrates basic product analytics concepts including event tracking, conversion funnel analysis, and user behavior analysis.

## Tools Used
- HTML
- Amplitude Analytics
- Loom (for walkthrough)
